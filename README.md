## To Do List

A simple console To Do List application written in Python.

This project was created as part of my Python learning journey.

## Features

- Three language support
  - Russian
  - English
  - German
- ➕ Create new tasks
- Display all tasks
- Delete tasks
- Console interface

## Technologies

- Python 3

## The application stores tasks in memory while it is running.
